# Rapport de Vérification de Conformité Légale - Immeuble Résidentiel à Casablanca (Maarif)

## 1. Introduction
Ce rapport présente les vérifications nécessaires pour confirmer la conformité légale d'un immeuble résidentiel situé à Casablanca, dans le quartier Maarif. Les aspects couverts incluent le zonage, les permis de construire, la réglementation marocaine, et d'autres exigences légales.

## 2. Vérifications Requises

### A. Zonage
- **Source** : Consulter le Plan d'Aménagement Urbain (PAU) de Casablanca.
- **Documents nécessaires** :
  - Extrait du plan de zonage.
  - Règles spécifiques (usage du sol, hauteur, densité).

### B. Permis de Construire
- **Source** : Direction Régionale de l'Urbanisme ou Commune de Casablanca.
- **Documents nécessaires** :
  - Copie du permis de construire.
  - Certificat de conformité.

### C. Conformité Réglementaire
- **Textes applicables** :
  - Loi n° 12-90 relative à l'urbanisme.
  - Loi n° 25-90 relative aux lotissements.
  - Code de la construction.

### D. Autres Exigences Légales
- **Titre de propriété** : Vérifier auprès du Conservateur de la Propriété Foncière.
- **Taxes** : Vérifier les droits d'enregistrement et les taxes communales.
- **Diagnostics** : État parasitaire et performance énergétique.

## 3. Risques Potentiels

| Risque                          | Description                                      | Action Recommandée                          |
|---------------------------------|--------------------------------------------------|---------------------------------------------|
| Non-conformité du zonage        | Usage du sol non conforme.                       | Obtenir un certificat d'urbanisme.         |
| Permis de construire invalide   | Permis manquant ou non conforme.                 | Vérifier le permis et le certificat.       |
| Défaut de titre de propriété    | Problème de traçabilité ou litige.                | Vérifier auprès du Conservateur.            |
| Taxes impayées                  | Dettes fiscales en suspens.                      | Exiger un état des taxes à jour.            |

## 4. Recommandations
1. Consulter un notaire pour vérifier le titre de propriété.
2. Obtenir tous les documents officiels (permis, zonage, taxes).
3. Effectuer une visite avec un expert pour vérifier la conformité.
4. Souscrire une assurance pour couvrir les risques.

## 5. Conclusion
Ce rapport met en évidence les vérifications nécessaires pour sécuriser l'achat. Une due diligence approfondie est essentielle pour éviter tout risque juridique ou financier.
